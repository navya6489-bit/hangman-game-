# Hangman Game (Python)

A simple command-line Hangman game built using Python.

## Features
- Random word with hint
- Tracks guesses
- ASCII hangman display
- Input validation

## Concepts Used
Functions, loops, lists, dictionaries, sets, random module

## How to Run
1. Install Python
2. Run:
   python hangman.py

## How to Play
Guess letters to find the word.
You get 6 wrong attempts before losing.
